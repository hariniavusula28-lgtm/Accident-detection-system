# AI-Based Accident Detection System

## Overview
This project is an AI-powered real-time accident detection system that uses Deep Learning and Computer Vision techniques to detect road accidents from live surveillance video streams.

The system uses CNN-LSTM architecture with TensorFlow and OpenCV for accident classification and alert generation.

---

## Features
- Real-time accident detection
- Video frame analysis using Deep Learning
- CNN-LSTM architecture
- Accident alert generation
- Computer Vision based monitoring
- Flask-based application
- TensorFlow integration

---
